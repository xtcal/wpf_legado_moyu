# 快捷键
- `Win + 小键盘4` 上一段
- `Win + 小键盘6` 下一段
- `Win + 小键盘5` 隐藏

# 功能
- [x] 置顶，透明框，拖拽，缩放，调整字号，调整阴影
- [x] 语音播报自动阅读（`System.Speech`）
- [x] 同步阅读章节进度
- [ ] 设置快捷键
- [ ] 拆分过长段落
